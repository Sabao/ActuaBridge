##最初に
 ActuaBridgeはActuaServerのコマンド及びステータス文字列を高速に送受信する為のプログラムです。  
 ActuaServerとの接続にはシリアル通信を用い、制御プログラム(ActuaClient)とのインターフェイスにはLinuxのメッセージキューを使います。
